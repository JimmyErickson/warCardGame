This game lets you play war (the card game) against the computer.
When the game ends, you can play again by pressing the "new cards" button.
The images should be placed in the following folders: cards (insode of) src.
If they are not there, they will not load into the game.
This game was written with swing in java with the eclipse IDE.
The game should not take any special processes to compile and run out side of
normal compiling and running through terminal.
